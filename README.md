# OpenRouter LLM Dashboard

This is a simple dashboard to interact with different Large Language Models (LLMs) from OpenRouter.

## Features

- Select from a list of available LLMs.
- Chat with the selected LLM.
- Responsive UI.

## Setup

1. Clone the repository.
2. Run `npm install` to install the dependencies.
3. Create a `.env` file and add your OpenRouter API key as `OPENROUTER_API_KEY`.
4. Run `npm start` to start the server.
5. Open `http://localhost:3000` in your browser.
